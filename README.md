# 🌱 Previsão de Safra (Satélite)

> Previsão de safra com imagens de satélite e ML. Monitora NDVI, estima produtividade e antecipa a colheita por talhão e região.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/previsao-safra/

[![Vertical](https://img.shields.io/badge/Vertical-Agro-blue)]() [![IA](https://img.shields.io/badge/IA-Computer%20Vision-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**< 4%** Erro safra · **2.8M ha** Área · **monitor.** Talhões · **60-75 dias** Antecip.

## 🧠 Tecnologias de IA
Computer Vision · Satellite · ML · NDVI

## ✨ O que faz
- NDVI por satélite (Sentinel-2)
- Estimativa de produtividade
- Previsão de colheita
- Detecção de estresse hídrico

## 🗄️ Fonte de dados real
[CONAB + Sentinel-2 (INPE)](https://www.conab.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com CONAB + Sentinel-2 (INPE)
2. **Processamento com IA** — Computer Vision · Satellite · ML · NDVI
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
